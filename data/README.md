# data/

Raw source data used to populate financial models.

**Contents:**
- Downloaded financial statements (annual reports, 10-K, audited filings)
- CSV exports from financial data portals (e.g., Cafef, Vietstock, Bloomberg, Macrotrends)

**Naming convention:** `{company}-{statement-type}-{YYYY}.{ext}`

Example: `mwg-income-statement-2023.csv`

> All data must be sourced from audited financial statements. Document the source URL in the filename or a companion `.md` note.
