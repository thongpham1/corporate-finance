# docs/

All written deliverables for the course, organized by type.

| Subfolder | Purpose |
|---|---|
| `decisions/` | Company selection memos and analytical decision records (Stage 2) |
| `specs/` | Technical specifications for LLM-driven analysis (Stage 4) |
| `plans/` | Optional project plans and timelines |

**Naming conventions:**
- Decision memos: `YYYY-MM-DD-pham-{company-slug}-selection.md`
- Specifications: `YYYY-MM-DD-pham-{company-slug}-spec.md`
- Use lowercase with hyphens for all filenames
