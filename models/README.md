# models/

All Excel work for the course.

| Subfolder | Purpose |
|---|---|
| `templates/` | Blank model frameworks provided by the instructor — do not modify |
| `builds/` | Populated models with actual company data |

**Naming convention:**
- Templates: `{model-name}-template.xlsx`
- Builds: `{company}-{model}-{YYYY-MM-DD}.xlsx`
